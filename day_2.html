What is Time Complexity?

Time complexity isn't about measuring the exact time an algorithm takes to run. Instead, it's about how the runtime of an algorithm grows as the size of the input grows. We use a notation called **Big O** to describe this relationship.

Think of it like this: if you double the input data, does the time it takes to run the algorithm also double? Does it quadruple? Or does it barely change at all? Big O notation gives us a standardized way to talk about these growth rates, helping us compare the efficiency of different solutions to a problem.
