Practice Problems

The final step is to apply what you've learned. The following problems are classic examples that can be solved efficiently using bit manipulation. Try to solve them on your own first, then click to reveal the C# solution and explanation. This will help solidify your understanding and prepare you for real-world interview questions.

1. Check if a number is even or odd.

Concept: An even number always has its least significant bit (LSB) as 0. An odd number always has it as 1. We can check this using the AND operator with a mask of 1.

public bool IsEven(int number) {
    return (number & 1) == 0;
}

2. Check if a number is a power of 2.

Concept: A positive number that is a power of 2 has only one bit set to 1 (e.g., 8 is 1000). If you subtract 1 from it, you get all bits set to 1 up to that position (e.g., 7 is 0111). ANDing these two numbers (n & (n-1)) will result in 0 if and only if n is a power of 2.

public bool IsPowerOfTwo(int n) {
    return n > 0 && (n & (n - 1)) == 0;
}

3. Count the number of set bits (1s).

Concept: A clever trick is to repeatedly use the expression `n & (n - 1)`. This operation unsets the rightmost set bit. We can count how many times we can do this until the number becomes 0.

public int CountSetBits(int n) {
    int count = 0;
    while (n > 0) {
        n &= (n - 1); // Unset the rightmost set bit
        count++;
    }
    return count;
}
