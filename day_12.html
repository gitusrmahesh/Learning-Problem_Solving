Day 12: Two Pointer Technique

What is the Two Pointer Technique?

The Two Pointer technique is an extremely common and efficient method for solving problems involving sorted arrays or linked lists. It involves using two pointers to iterate through the data structure, moving them based on certain conditions to find a solution. This approach can often reduce the time complexity from a brute-force O(n²) to a much more optimal O(n).

The Core Idea

Instead of using nested loops, we initialize two pointers, often at opposite ends of a sorted array. We then move them towards each other or in the same direction, narrowing down the search space with each step until we find the solution or the pointers meet/cross.

Problem: Find a Pair with a Given Sum

Given a sorted array of integers `nums` and an integer `target`, return `true` if there are two numbers in the array that add up to `target`, otherwise return `false`.

Interactive Visualizer

C# Solution

public bool HasPairSum(int[] nums, int target) { if (nums == null || nums.Length < 2) { return false; } int left = 0; int right = nums.Length - 1; while (left < right) { int currentSum = nums[left] + nums[right]; if (currentSum == target) { return true; } else if (currentSum < target) { left++; } else // currentSum > target { right--; } } return false; }

Problem: Find a Triplet with a Given Sum

Given a sorted array of integers `nums` and an integer `target`, return `true` if there are three numbers in the array that add up to `target`, otherwise return `false`.

Interactive Visualizer

Log:

C# Solution

public bool HasTripletSum(int[] nums, int target) { if (nums == null || nums.Length < 3) { return false; } Array.Sort(nums); for (int i = 0; i < nums.Length - 2; i++) { int left = i + 1; int right = nums.Length - 1; int newTarget = target - nums[i]; while (left < right) { int currentSum = nums[left] + nums[right]; if (currentSum == newTarget) { return true; } else if (currentSum < newTarget) { left++; } else { right--; } } } return false; }

Problem: Remove Duplicates from Sorted Array

Given a sorted array `nums`, remove the duplicates in-place such that each unique element appears only once. Return the new length of the array.

Interactive Visualizer

Log:

C# Solution

public int RemoveDuplicates(int[] nums) { if (nums.Length == 0) { return 0; } // 'i' is the slow-runner pointer int i = 0; // 'j' is the fast-runner pointer for (int j = 1; j < nums.Length; j++) { if (nums[j] != nums[i]) { i++; nums[i] = nums[j]; } } // The new length is i + 1 return i + 1; }
