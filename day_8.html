1. Array Traversal

Traversal means visiting or accessing each element of an array exactly once. This is the most fundamental array operation, commonly done using a `for` or `foreach` loop in C# to read, process, or display elements.

Interactive Demo

C# Code Example (`for` loop)

int[] numbers = { 10, 20, 30, 40, 50 }; Console.WriteLine("Array elements:"); for (int i = 0; i < numbers.Length; i++) { // Accessing element at index i Console.WriteLine(numbers[i]); }

2. Array Insertion

Since C# arrays have a fixed size, you cannot directly "insert" an element into a full array. The process involves creating a new, larger array and copying elements from the original array, along with the new element, into their correct positions.

Practice: Array Rotation

Array rotation is a common operation where the elements of an array are shifted by a certain number of positions. A left rotation shifts elements to the beginning of the array, while a right rotation shifts them towards the end. Elements that are shifted off one end reappear on the other.
